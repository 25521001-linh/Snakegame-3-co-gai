**Cách chấm điểm:**

a. Tăng điểm:
+ Ăn 1 mồi bình thưởng = +1 điểm, tăng độ dài, tăng tốc độ
+ Ăn 1 mồi đặc biệt = +5 điểm
  
b. Thua:
+ Rắn đụng vào thân mình = mất hết điểm => thua

**Mồi:**

a. Mồi thường:
+ Thời gian xuất hiện: ngay lập tức
+ Vị trí xuất hiện: ngay lập tức
+ Chức năng: tăng điểm, độ dài, tốc độ

b. Mồi đặc biệt:
+ Thời gian xuất hiện: khi tăng 5 điểm
+ Vị trí xuất hiện: trung tâm bản đồ, có chướng ngại vật bao quanh
+ Chức năng: tăng 5 điểm
